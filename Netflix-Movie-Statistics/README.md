# Netflix Movie Statistics Project

## Overview
This project analyzes Netflix Originals movie data using Python.

The analysis includes:
- Checking missing and duplicate values
- Language distribution
- IMDB score distribution
- Genre distribution
- Average IMDB score by genre
- Runtime distribution
- Outlier detection using IQR
- Relationship between runtime and language
- Correlation between runtime and IMDB score

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
NetflixOriginals.csv

## Key Insights
- English dominates Netflix Originals.
- IMDB scores are mostly concentrated around the middle range.
- Some genres have higher average ratings.
- Runtime contains outliers.
- Runtime and IMDB score show weak correlation.
